# 台鐵職工福利平台 — SQL 資料庫

本目錄採用**雙資料庫**架構，同時支援 **MySQL 8.x** 與 **PostgreSQL 15+**。

## 目錄結構

```
sql/
├── mysql/                # MySQL 版本（原始版本）
│   ├── modules/          # 22 個模組 DDL
│   ├── tests/            # 測試 SQL
│   ├── scripts/          # 拼接與驗證腳本
│   ├── all_in_one.sql    # 整合檔案
│   ├── README_SQL.md     # MySQL 說明文件
│   └── tra_welfare_platform.sql  # 保留舊版基線
│
├── postgresql/           # PostgreSQL 版本（轉換版本）
│   ├── modules/          # 22 個模組 DDL
│   ├── tests/            # 測試 SQL
│   ├── scripts/          # 拼接與驗證腳本
│   └── README_PG.md      # PostgreSQL 說明文件
│
└── convert_mysql_to_pg.py  # MySQL→PostgreSQL 轉換腳本
```

## 使用方式

- **MySQL**：請見 `sql/mysql/README_SQL.md`
- **PostgreSQL**：請見 `sql/postgresql/README_PG.md`

## 轉換範圍

| 項目 | 數量 |
|------|------|
| CREATE TABLE | ~189 張 |
| CREATE FUNCTION | 1 個 |
| CREATE PROCEDURE | 7 個 |
| CREATE TRIGGER | 10 個 |
| CREATE VIEW | 7 個 |

## 主要轉換項目

- `AUTO_INCREMENT` → `GENERATED BY DEFAULT AS IDENTITY`
- `TINYINT(1)` → `BOOLEAN`
- `DATETIME` → `TIMESTAMP`
- `JSON` → `JSONB`
- `ENGINE=InnoDB` / `CHARSET=utf8mb4` → 移除
- `KEY` / `UNIQUE KEY` → `INDEX` / `UNIQUE`
- `ON UPDATE CURRENT_TIMESTAMP` → 觸發器實作（待補）
