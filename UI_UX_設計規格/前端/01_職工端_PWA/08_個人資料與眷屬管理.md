# 個人資料與眷屬管理

## 1. 功能概述

職工檢視與維護個人基本資料、聯絡資訊，以及管理眷屬資料（新增/編輯/刪除）。

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 員工編號 | Text | employee.employee_no |
| 姓名 | Text | employee.full_name |
| 所屬單位 | Text | org_node.node_name (透過 welfare_branch_id) |
| 在職狀態 | StatusBadge | employee.employment_status |
| 人員類型 | Badge | employee.staff_type |
| 電話 Input | Input | employee.phone |
| Email Input | Input | employee.email |
| 地址 Input | Input | employee.address |
| 儲存按鈕 | Button | PUT /employees/{id} (含 row_version) |
| 扣繳狀態 | Badge | employee.payroll_deduction_status |
| 補助資格 | Badge | employee.subsidy_eligibility_status |
| 眷屬姓名 | Text | employee_dependent.dependent_name |
| 眷屬關係 | Badge | employee_dependent.relation_type |
| 眷屬補助資格 | Badge | employee_dependent.is_eligible_for_subsidy |
| 編輯眷屬 Button | Button | 彈出 Dialog |
| 刪除眷屬 Button | Button | ConfirmDialog |
| 新增眷屬 Button | Button | 彈出 Dialog |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<Card>` | 各資訊區塊 | 基本/聯絡/資格/眷屬 |
| `<Input>` | 編輯聯絡資訊 | 編輯模式才可編輯 |
| `<Button>` | 儲存/編輯/刪除 | variant 區分 |
| `<Badge>` | 狀態/關係標籤 | StatusBadge 封裝 |
| `<Dialog>` | 新增/編輯眷屬表單 | 內嵌 Form |
| `<Form>` + `<FormField>` | 眷屬資料表單 | react-hook-form |
| `<Select>` | 眷屬關係選擇 | spouse/child/parent/other |
| `<DatePicker>` | 出生日期 | popover 日曆 |
| `<ConfirmDialog>` (自訂) | 刪除確認 | AlertDialog 模式 |
| `<Toast>` | 儲存成功/失敗 | Sonner |

## 5. 互動與狀態

| 狀態 | 處理方式 |
|------|----------|
| Loading | Skeleton 填入各區塊 |
| Edit mode | 聯絡資訊 Input 啟用編輯，Button 變「取消」「儲存」 |
| Error - 儲存失敗 | Alert「儲存失敗，請稍後再試」 |
| Success | Toast「資料已更新」 |
| Edge - 身分證欄位 | 僅顯示遮罩值，不可編輯 |
| Edge - 眷屬刪除 | ConfirmDialog「確認刪除此眷屬資料？」 |

## 6. 權限控管

- 僅限本人檢視與編輯本人的資料與眷屬
- 身分證字號僅顯示遮罩值，不可修改
- 員工編號、姓名、單位等核心欄位不可編輯

## 7. 相關頁面與路由

- 個人資料：/profile
- 眷屬管理：/profile/dependents（可獨立路由或 Tab）
