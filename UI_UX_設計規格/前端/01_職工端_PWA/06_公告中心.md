# 公告中心

## 1. 功能概述

職工瀏覽系統公告與福利規章文件。支援分類篩選、置頂公告、公告詳情閱讀與規章文件下載。

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 分類篩選標籤 | Tabs | announcement_category.category_code |
| 公告卡片 | Card | announcement 主表欄位 |
| 置頂標記 | Badge | announcement.is_pinned |
| 公告分類 Badge | Badge | announcement_category.category_name |
| 公告日期 | Text | announcement.publish_start_at |
| 瀏覽次數 | Text | announcement.view_count |
| 公告內容 | HTML Render (dangerouslySetInnerHTML) | announcement.content |
| 規章文件列表 | Card + DownloadButton | policy_document |
| 分頁 | Pagination | GET /announcements?page |
| 下載按鈕 | Button | GET /files/{fileId}/download |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<Tabs>` | 分類篩選 | 全部/福利資訊/系統公告/活動訊息 |
| `<Card>` | 公告列表項 | 標題+日期+分類 |
| `<Badge>` | 分類/置頂標記 | 置頂用 variant="accent" |
| `<Pagination>` | 分頁 | 每頁 10 筆 |
| `<Button>` | 下載規章 | variant="outline" + 下載圖示 |
| `<Skeleton>` | 載入中 | 列表/詳情 |
| `<Separator>` | 列表分隔線 | - |

## 5. 互動與狀態

| 狀態 | 處理方式 |
|------|----------|
| Loading | Skeleton × 5 |
| Empty - 無公告 | 「目前暫無公告」 |
| Empty - 篩選結果 | 「無此分類的公告」 |
| Error | Alert + 重試按鈕 |
| Edge - 已過期公告 | 自動隱藏 (publish_end_at 判斷) |

## 6. 權限控管

- 所有已登入職工均可瀏覽公告與下載規章
- 公告依 audience_scope 過濾可見性（region/branch/role/all）

## 7. 相關頁面與路由

- 公告列表：/announcements
- 公告詳情：/announcements/[id]
- 規章文件：/policies（可合併或獨立路由）
