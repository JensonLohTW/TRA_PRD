# 掃描規則與執行紀錄

## 1. 功能概述

管理自動化稽核掃描規則（啟用/停用/設定），並檢視掃描執行歷程與結果。

## 2. 頁面架構

### 掃描規則（/security/rules）

```
+------------------------------------------+
|  掃描規則                          [+新增] |
+------------------------------------------+
|  ┌──────┬──────┬──────┬──────┬──────┬──┐  |
|  │規則代碼│名稱  │類別  │啟用  │排程  │操作│  |
|  ├──────┼──────┼──────┼──────┼──────┼──┤  |
|  │SCAN01│異常  │login │✅   │每 5m │[編輯]│
│  │      │登入  │      │      │      │    │  |
|  │SCAN02│大量  │file  │✅   │每日  │[編輯]│
│  │      │下載  │      │      │02:00 │    │  |
|  └──────┴──────┴──────┴──────┴──────┴──┘  |
+------------------------------------------+
```

### 掃描執行紀錄（/security/scan-runs）

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 規則列表 | DataTable | security_scan_rule |
| 規則編輯 Dialog | Dialog | rule_config_json / schedule_expr |
| 啟用 Switch | Switch | is_enabled |
| 掃描紀錄列表 | DataTable | security_scan_run |
| 掃描詳情 | Dialog | summary_text / findings_count |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<DataTable>` (自訂) | 規則/紀錄列表 | - |
| `<Dialog>` | 規則編輯/掃描詳情 | - |
| `<Form>` | 規則表單 | - |
| `<Switch>` | 啟用/停用 | - |
| `<Badge>` | 執行狀態 | - |
| `<Button>` | 新增/編輯/檢視 | - |

## 5. 相關頁面與路由

- 掃描規則：/security/rules
- 掃描紀錄：/security/scan-runs
- 掃描詳情：/security/scan-runs/[id]
