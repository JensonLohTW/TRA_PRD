# 資安報表與文件庫

## 1. 功能概述

管理資安報告與文件，產製掃描報告、滲透測試報告、合規報告等，並管理資安文件庫。

## 2. 頁面架構

### 資安報表（/security/reports）

```
+------------------------------------------+
|  資安報表                          [+產製] |
+------------------------------------------+
|  [掃描報告] [合規報告] [稽核摘要] [封存匯出] |
+------------------------------------------+
|  ┌──────┬──────┬──────┬──────┬──────┬──┐  |
|  │名稱  │類型  │日期  │風險  │下載  │操作│  |
|  ├──────┼──────┼──────┼──────┼──────┼──┤  |
|  │06月  │掃描  │06/30 │安全  │25次  │[檢視]│
|  │掃描  │      │      │      │      │    │  |
|  │Q2    │合規  │06/30 │中風險│8次   │[檢視]│
|  │合規  │      │      │      │      │    │  |
|  └──────┴──────┴──────┴──────┴──────┴──┘  |
+------------------------------------------+
```

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 報表分類 Tabs | Tabs | report_category |
| 報表列表 | DataTable | security_report_library |
| 風險標籤 | Badge | risk_label |
| 產製報表 Button | Button | POST /security/reports/generate |
| 下載 Button | Button | GET /security/reports/{id}/download |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<Tabs>` | 報表分類切換 | - |
| `<DataTable>` (自訂) | 報表列表 | - |
| `<Badge>` | 風險標籤 | safe/medium/high |
| `<Button>` | 產製/下載 | - |

## 5. 相關頁面與路由

- 資安報表：/security/reports
- 報表詳情：/security/reports/[id]
