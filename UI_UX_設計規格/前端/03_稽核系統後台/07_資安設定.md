# 資安設定

## 1. 功能概述

管理資安相關系統設定，含告警閾值、通知對象、日誌保留天數等。

## 2. 頁面架構

```
+------------------------------------------+
|  資安設定                                 |
+------------------------------------------+
|  ┌── 告警閾值 ────────────────────────┐  |
|  │   登入失敗次數上限：[5]              │  |
|  │   OTP 錯誤次數上限：[5]              │  |
|  │   大量下載觸發次數：[100/hr]         │  |
|  │   [儲存]                            │  |
|  └────────────────────────────────────┘  |
|                                          |
|  ┌── 通知對象 ────────────────────────┐  |
|  │   Email：[admin@tra.gov.tw]         │  |
|  │   [新增通知對象]                     │  |
|  │   ┌──────┬──────┬──────┬──┐       │  |
|  │   │管道  │對象  │啟用  │操作│       │  |
|  │   ├──────┼──────┼──────┼──┤       │  |
|  │   │Email │admin │✅   │[刪除]│       │  |
|  │   └──────┴──────┴──────┴──┘       │  |
|  └────────────────────────────────────┘  |
|                                          |
|  ┌── 日誌設定 ────────────────────────┐  |
|  │   熱區保留天數：[90]                 │  |
|  │   冷區保留年數：[3]                  │  |
|  │   [儲存]                            │  |
|  └────────────────────────────────────┘  |
+------------------------------------------+
```

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 告警閾值 Form | Form | sys_config (config_key prefix: alert.*) |
| 通知對象列表 | Table | scan_rule_notify_target |
| 通知對象 Dialog | Dialog | 新增 channel/target |
| 日誌設定 Form | Form | sys_config (config_key prefix: audit.*) |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<Form>` | 閾值/日誌設定 | - |
| `<Input type="number">` | 數值輸入 | - |
| `<Table>` | 通知對象列表 | - |
| `<Dialog>` | 新增通知對象 | - |
| `<Select>` | 管道選擇 | email/system_message |
| `<Button>` | 儲存/新增/刪除 | - |
| `<Switch>` | 啟用/停用 | - |

## 5. 權限控管

- 僅系統管理者可存取資安設定

## 6. 相關頁面與路由

- 資安設定：/security/settings
