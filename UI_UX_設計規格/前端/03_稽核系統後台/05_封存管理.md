# 封存管理

## 1. 功能概述

管理稽核日誌的封存包（熱冷分區），檢視封存包列表、產製封存包、下載封存包。確保日誌保存至少 3 年。

## 2. 頁面架構

```
+------------------------------------------+
|  封存管理                           [+產製] |
+------------------------------------------+
|  ┌──────┬──────┬──────┬──────┬──────┬──┐  |
|  │封存標籤│日期  │筆數  │大小  │狀態  │操作│  |
|  ├──────┼──────┼──────┼──────┼──────┼──┤  |
|  │2026Q2│06/30 │12,500│45.2MB│已封存│[下載]│
|  │2026Q1│03/31 │10,200│38.1MB│已封存│[下載]│
|  │2025Q4│01/01 │8,500 │30.5MB│已封存│[下載]│
|  └──────┴──────┴──────┴──────┴──────┴──┘  |
+------------------------------------------+
```

### 封存包詳情（/security/archives/[id]）

```
+------------------------------------------+
|  ← 封存管理    封存包 2026Q2              |
+------------------------------------------+
|  封存資訊                                |
|  標籤：2026Q2                             |
|  日期：2026/06/30                         |
|  事件筆數：12,500                         |
|  檔案大小：45.2 MB                        |
|  檔案類型：.enc (加密封存)                |
|  雜湊值：SHA-256: a3f8...                |

|  下載次數：3                              |
|                                          |
|  [下載封存包]  [驗證完整性]               |
+------------------------------------------+
```

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 封存列表 | DataTable | audit_archive_bundle |
| 產製 Button | Button | POST /security/archives/generate |
| 封存詳情 | Card | 完整欄位 |
| 下載 Button | Button | GET /security/archives/{id}/download |
| 驗證 Button | Button | POST /security/archives/{id}/verify |
| 下載次數 | Text | download_count |
