# 稽核日誌查詢

## 1. 功能概述

查詢全系統操作軌跡日誌，支援多維度篩選、日誌詳情檢視、匯出。日誌保存至少 3 年，具防竄改設計。

## 2. 頁面架構

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 時間區間 Pickers | DateRangePicker | event_time BETWEEN |
| 操作者搜尋 | Combobox | actor_employee_id |
| 動作 Select | Select | action_code |
| 目標類型 Select | Select | target_type |
| 嚴重程度 Select | Select | severity_level |
| 關鍵字 Input | Input | detail_json::text LIKE |
| 查詢/重置 Button | Button | - |
| 日誌表格 | DataTable | system_audit_trail |
| 日誌詳情 | Card + CodeBlock | system_audit_trail 完整欄位 |
| 匯出 Button | Button | GET /security/audit-logs/export |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<DataTable>` (自訂) | 日誌列表 | 時間/操作者/動作/目標 |
| `<SearchFilterBar>` (自訂) | 多維度篩選 | 時間/操作者/動作/嚴重度 |
| `<DateRangePicker>` (自訂) | 時間區間 | - |
| `<Combobox>` (自訂) | 操作者搜尋 | - |
| `<Select>` | 動作/目標/嚴重度 | - |
| `<Input>` | 關鍵字 | - |
| `<Button>` | 查詢/重置/匯出 | - |
| `<Card>` | 日誌詳情 | - |
| `<CodeBlock>` (自訂) | JSON 明細 | syntax highlight |
| `<Pagination>` | 分頁 | 每頁 50 筆 |
| `<Badge>` | 嚴重程度 | variant 映射 |
