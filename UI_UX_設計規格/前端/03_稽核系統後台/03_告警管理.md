# 告警管理

## 1. 功能概述

管理資安告警，含告警列表、告警詳情、告警分派、處置與結案。告警來源為自動化稽核掃描與即時規則引擎。

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 告警列表 | DataTable | security_alert |
| 狀態篩選 | Tabs | alert_status |
| 嚴重程度 Badge | Badge | severity_level |
| 告警詳情 | Card | security_alert 完整欄位 |
| 事件明細 | CodeBlock | audit_event (關聯) |
| 分派 Select | Select | assigned_to |
| 處理說明 Textarea | Textarea | resolution_note |
| 分派 Button | Button | PUT /security/alerts/{id}/assign |
| 已知悉 Button | Button | PUT /security/alerts/{id}/acknowledge |
| 結案 Button | Button | PUT /security/alerts/{id}/resolve |
