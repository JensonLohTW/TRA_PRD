# 稽核儀表板與登入

## 1. 功能概述

稽核後台登入採強制 MFA。儀表板展示系統安全概覽：告警統計、近期告警、日誌寫入量趨勢、系統健康度。

## 2. 頁面架構

### 資安儀表板（/security）

```
+------------------------------------------+
|  資安儀表板                       [使用者] |
+------------------------------------------+
|  告警統計                                |
|  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐   |
|  │待處理│ │處理中│ │已解決│ │今日  │ │本週  │   |
|  │ 8   │ │ 3   │ │ 15  │ │新增 3│ │新增 12│  |
|  └────┘ └────┘ └────┘ └────┘ └────┘   |
+------------------------------------------+
|  近期告警 (高/嚴重)                        |
|  ┌──────────────────────────────────────┐ |
|  │ 🔴 異常登入嘗試 - 臺北區    10 分鐘前 │ |
|  │ 🟡 大量檔案下載 - EMP015   30 分鐘前 │ |
|  │ 🔴 權限異動 - 非上班時間   1 小時前  │ |
|  └──────────────────────────────────────┘ |
+------------------------------------------+
|  日誌寫入量趨勢 (本週)                    |
|  ┌──────────────────────────────────────┐ |
|  │ [LineChart: 每日事件數]               │ |
|  └──────────────────────────────────────┘ |
+------------------------------------------+
```

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 告警統計卡 | StatCard | GET /security/dashboard/alert-statistics |
| 近期告警列表 | Card | GET /security/alerts?limit=5&severity=high,critical |
| 日誌趨勢圖 | LineChart | GET /security/dashboard/audit-trend |
| 系統公告 | Card | - |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<StatCard>` (自訂) | 告警統計 | - |
| `<Card>` | 近期告警 | 嚴重程度標色邊框 |
| `<LineChart>` (Recharts) | 日誌趨勢 | - |
| `<Skeleton>` | 載入中 | - |

## 5. 權限控管

- 稽核者：檢視儀表板與告警
- 系統管理者：完整權限

## 6. 相關頁面與路由

- 登入：/security/login
- 儀表板：/security
- 點選告警 → /security/alerts/[id]
