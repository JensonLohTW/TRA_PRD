# 財務傳票管理

## 1. 功能概述

依核准的撥款批次產製收入/支出傳票初稿、代請款報銷單及核定名冊，格式與機關現行紙本對齊，供財務人員校對後輸出。

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 傳票列表 | DataTable | ai_voucher_generation / voucher (PG) |
| 傳票內容 | Table | voucher_line (借貸明細) |
| 借貸平衡狀態 | Badge | 計算借方總和 = 貸方總和 |
| 關聯案件列表 | Table | payment_batch_item (與 application 關聯) |
| 確認 Button | Button | POST /payment/vouchers/{id}/confirm |
| 輸出 PDF Button | Button | GET /payment/vouchers/{id}/export-pdf |
| 輸出 Excel Button | Button | GET /payment/vouchers/{id}/export-excel |
| 產製傳票 Button | Button | POST /payment/batches/{id}/generate-voucher |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<DataTable>` (自訂) | 傳票列表 | - |
| `<Table>` | 傳票明細/科目 | 借貸方格式 |
| `<Card>` | 傳票內容容器 | - |
| `<Badge>` | 借貸平衡/狀態 | variant 區分 |
| `<Button>` | 確認/輸出 | variant="default" / "outline" |
| `<SearchFilterBar>` (自訂) | 篩選 | 批次/狀態 |

## 5. 權限控管

- 僅財務角色可檢視與操作傳票
- 傳票確認後不可直接修改，需依作廢流程處理

## 6. 相關頁面與路由

- 傳票列表：/admin/payment/vouchers
- 傳票校對：/admin/payment/vouchers/[id]
- 從批次詳情產製：/admin/payment/batches/[id] → [產製傳票]
