# 系統設定與字典管理

## 1. 功能概述

管理端維護系統全域配置，含資料字典管理、系統參數設定。字典治理業務流程狀態與可配置選項。

## 2. 頁面架構

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 字典類型 Select | Select | sys_dictionary.dict_type (distinct) |
| 字典值表格 | DataTable | sys_dictionary |
| 字典值編輯 Dialog | Dialog | dict_code / dict_name / sort_order |
| 系統參數表格 | DataTable | sys_config |
| 參數編輯 Dialog | Dialog | config_value / is_editable |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<Select>` | 字典類型選擇 | - |
| `<DataTable>` (自訂) | 字典值/參數列表 | - |
| `<Dialog>` | 新增/編輯彈窗 | - |
| `<Form>` | 編輯表單 | - |
| `<Input>` | 代碼/名稱/值 | - |
| `<Switch>` | 啟用/停用 | - |
| `<Button>` | 操作按鈕 | - |

## 5. 權限控管

- 僅系統管理者可操作
- 重要參數異動留存稽核軌跡

## 6. 相關頁面與路由

- 系統參數：/admin/system/config
