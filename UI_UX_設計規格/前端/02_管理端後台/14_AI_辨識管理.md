# AI 辨識管理

## 1. 功能概述

管理端檢視 AI OCR 辨識結果，支援人工校正、異常處理。OCR 為非阻塞輔助功能，不阻斷主流程。

## 2. 頁面架構

### 辨識結果列表（/admin/ocr）

```
+------------------------------------------+
|  AI 辨識管理                              |
+------------------------------------------+
|  [全部] [待校正] [已完成] [異常]            |
+------------------------------------------+
|  ┌──────┬──────┬──────┬──────┬──────┬──┐  |
|  │申請案號│文件類型│信心度│狀態  │上傳日│操作│  |
|  ├──────┼──────┼──────┼──────┼──────┼──┤  |
|  │TP..  │戶口名簿│0.95  │已完成│06/20│[檢視]│
|  │TP..  │診斷證明│0.45  │待校正│06/19│[校正]│
|  │TP..  │收據   │0.30  │異常  │06/18│[校正]│
|  └──────┴──────┴──────┴──────┴──────┴──┘  |
+------------------------------------------+
```

### 辨識校正（/admin/ocr/[id]）

```
+------------------------------------------+
|  ← AI 辨識管理    診斷證明校正             |
+------------------------------------------+
|  ┌── 原始影像 ──────┐  ┌── OCR 結果 ──┐  |
|  │ [影像預覽]        │  │ 欄位   信心度  │  |
|  │                   │  │ 金額   0.45  │  |
|  │                   │  │ 日期   0.80  │  |
|  │                   │  │ 醫院   0.92  │  |
|  └───────────────────┘  │              │  |
|                          │ [全部接受]   │  |
|                          │ [手動修正]   │  |
|  ┌── 人工校正 ────────────────────┐   │  |
|  │  金額：[12,000]    修正說明：[..]  │   │  |
|  │  [儲存校正]                      │   │  |
|  └────────────────────────────────┘   │  |
+------------------------------------------+
```

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 辨識列表 | DataTable | ai_document_recognition |
| 狀態篩選 | Tabs | recognition_status |
| 原始影像 | Image | file_resource.storage_path |
| OCR 結果表 | Table | recognized_fields_json |
| 信心度 Progress | Progress | confidence_score |
| 人工校正 Form | Form | corrected_fields_json |
| 儲存校正 Button | Button | PUT /ocr/{id}/correct |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<DataTable>` (自訂) | 辨識列表 | - |
| `<Tabs>` | 狀態篩選 | - |
| `<Image>` | 原始影像預覽 | 可放大 |
| `<Table>` | OCR 欄位+信心度 | 信心度低標紅 |
| `<Progress>` | 信心度進度條 | color 依數值變色 |
| `<Form>` | 人工校正表單 | - |
| `<Input>` | 校正值 | - |
| `<Textarea>` | 校正說明 | - |
| `<Button>` | 接受/儲存 | - |

## 5. 權限控管

- 承辦人：檢視所轄案件 OCR 結果、執行校正
- 系統管理者：全域檢視

## 6. 相關頁面與路由

- AI 辨識列表：/admin/ocr
- 辨識校正：/admin/ocr/[id]
- 從案件詳情亦可連至此頁
