# 管理儀表板與登入

## 1. 功能概述

管理端登入採強制 MFA（OTP + 密碼）。儀表板提供全署案件統計摘要、待辦事項、近期活動與系統健康度概覽。

## 2. 頁面架構

### 登入頁（/admin/login）

```
+------------------------------------------+
|           臺鐵職福會管理後台               |
+------------------------------------------+
|  帳號：[________________]                 |
|  密碼：[________________]                 |
|  [登入]                                  |
|  忘記密碼？                               |
+------------------------------------------+
```

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 帳號 Input | Input | user_account.login_name |
| 密碼 Input | Input (password) | user_account.password_hash |
| MFA OTP Input | OTPInput | 二次驗證 |
| 統計卡 | StatCard | GET /admin/dashboard/statistics |
| 待辦列表 | Card | GET /admin/review-tasks?limit=5 |
| 進度圖表 | BarChart (Recharts) | GET /admin/dashboard/chart |
| 系統公告 | Card | GET /admin/announcements?limit=3&scope=system |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<Card>` | 登入表單容器 | 置中 |
| `<Input>` | 帳號 | - |
| `<Input type="password">` | 密碼 | 顯示/隱藏切換 |
| `<Button>` | 登入 | loading |
| `<StatCard>` (自訂) | 統計數字 | onClick 導向對應列表 |
| `<Card>` | 待辦事項 | 點選導向待辦 |
| `<BarChart>` (Recharts) | 申請量趨勢 | 本週/本月 |
| `<Skeleton>` | 載入中 | 各區塊獨立 |

## 5. 權限控管

- 管理端所有頁面需 MFA 驗證
- 儀表板內容依角色過濾（承辦人看審核統計，財務看撥款統計）

## 6. 相關頁面與路由

- 登入：/admin/login
- 儀表板：/admin
- 點選統計卡 → 過濾後的對應列表頁
