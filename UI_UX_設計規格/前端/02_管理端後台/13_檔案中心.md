# 檔案中心

## 1. 功能概述

管理端查詢所有上傳檔案，檢視檔案元資料、下載審計紀錄，支援檔案預覽與下載。作為 M08 檔案資源中心的管理介面。

## 2. 頁面架構

```
+------------------------------------------+
|  檔案中心                          [匯出] |

|  篩選列                                  |

|  敏感等級：[全部 v]   狀態：[全部 v]        |
|  關鍵字：[________]                       |
+------------------------------------------+
|  ┌──────┬──────┬──────┬──────┬──────┬──┐  |
```

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 檔案列表 | DataTable | file_resource |
| 篩選列 | SearchFilterBar | business_type / sensitivity_level / file_status |
| 檔案預覽 Dialog | Dialog | file_resource.storage_path |
| 下載 Button | Button | GET /files/{id}/download |
| 下載紀錄表 | Table | file_download_log |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<DataTable>` (自訂) | 檔案列表 | - |
| `<SearchFilterBar>` (自訂) | 篩選 | - |
| `<Dialog>` | 檔案預覽 | 圖片/PDF 內嵌 |
| `<Button>` | 下載/預覽 | - |
| `<Table>` | 下載紀錄 | - |
| `<Badge>` | 敏感等級標記 | variant 區分 |
| `<Pagination>` | 分頁 | - |

## 5. 權限控管

- 依角色過濾檔案可見性（敏感等級限制）
- 下載行為自動寫入 file_download_log
