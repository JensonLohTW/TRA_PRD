# 通知模板管理

## 1. 功能概述

管理端維護通知模板（portal/email/sms），設定模板內容與變數 schema。供 M09 通知中心發送時套用。

## 2. 頁面架構

```
+------------------------------------------+
|  通知模板管理                       [+新增] |
+------------------------------------------+
|  ┌──────┬──────┬──────┬──────┬─────┬──┐  |
|  │模板代碼│名稱  │管道  │啟用  │變數  │操作│  |
|  ├──────┼──────┼──────┼──────┼─────┼──┤  |
|  │APP.. │申請  │portal│✅   │3    │[編輯]│
|  │      │送審  │      │      │     │    │  |
|  │APP.. │申請  │email │✅   │3    │[編輯]│
|  │      │核准  │      │      │     │    │  |
|  └──────┴──────┴──────┴──────┴─────┴──┘  |
+------------------------------------------+
```

### 模板編輯

```
+------------------------------------------+
|  ← 通知模板    編輯「申請送審通知」         |
+------------------------------------------+
|  模板代碼：[APP_SUBMIT]                   |
|  模板名稱：[申請送審通知]                   |
|  管道：[portal v]                          |
|                                          |
|  主旨範本：                                |
|  ┌──────────────────────────────────────┐ |
|  │ {{applicant_name}} 的 {{benefit_name} │ |
|  │ } 補助申請已送審                      │ |
|  └──────────────────────────────────────┘ |
|                                          |
|  內容範本：                                |
|  ┌──────────────────────────────────────┐ |
|  │ 親愛的 {{applicant_name}} 您好，      │ |
|  │ 您的 {{benefit_name}} 補助申請       │ |
|  │ (案號：{{application_no}}) 已成功    │ |
|  │ 送審，目前處理中。                    │ |
|  └────────────────────────────────────┘  |
|                                          |
|  變數清單：                                |
|  applicant_name, benefit_name,            |
|  application_no, submitted_at             |
|                                          |
|  [測試發送]  [儲存]                       |
+------------------------------------------+
```

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 模板列表 | DataTable | notification_template |
| 管道 Select | Select | channel (portal/email/sms) |
| 主旨 Input | Input | subject_template |
| 內容 Textarea | Textarea | body_template |
| 變數 schema | CodeBlock | variable_schema_json |
| 測試發送 Button | Button | POST /notification-templates/{id}/test |
| 儲存 Button | Button | PUT /notification-templates/{id} |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<DataTable>` (自訂) | 模板列表 | - |
| `<Form>` | 模板編輯 | - |
| `<Input>` | 主旨/代碼/名稱 | - |
| `<Textarea>` | 內容範本 | 支援 {{variable}} |
| `<Select>` | 管道選擇 | - |
| `<Badge>` | 啟用狀態 | - |
| `<Button>` | 測試/儲存 | - |

## 5. 權限控管

- 僅系統管理者可編輯通知模板
- 模板異動影響自動通知內容

## 6. 相關頁面與路由
