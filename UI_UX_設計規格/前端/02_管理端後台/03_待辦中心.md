# 待辦中心

## 1. 功能概述

管理端統一的待辦任務工作台，依角色顯示待審核、待簽核的補助案件與發款批次等任務。支援任務分類篩選、排序與快速審批操作。

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 統計摘要 | StatCard | GET /review-tasks/statistics |
| 類型篩選 | Select | review_task.business_type |
| 急迫篩選 | Select | review_task.urgency_level |
| 待辦卡片 | Card | review_task + benefit_application 摘要 |
| 急迫程度標記 | Badge | review_task.urgency_level |
| 到期天數 | Text | review_task.due_at (diff) |
| 立即處理 Button | Button | → 待辦詳情 or 直接跳案件詳情 |
| 案件摘要 | Card | benefit_application (唯讀) |
| 審批操作區 | Form + Button | POST /ben/applications/{id}/approve |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<StatCard>` (自訂) | 統計摘要 | 待處理/急迫/逾期 |
| `<Select>` | 類型/急迫篩選 | - |
| `<Card>` | 待辦卡片 | 急迫程度標色邊框 |
| `<Badge>` | 急迫標記 | critical=red, urgent=yellow, normal=gray |
| `<Pagination>` | 分頁 | - |
| `<Button>` | 立即處理 | variant="default" |
| `<Textarea>` | 審核意見 | - |
| `<Button>` | 核准/退回/駁回 | variant 區分 |
| `<ConfirmDialog>` (自訂) | 審批確認 | - |

## 5. 互動與狀態

| 狀態 | 處理方式 |
|------|----------|
| Loading | Skeleton × 5 |
| Empty | 「目前無待辦任務」 |
| Error | Alert + 重試 |
| Edge - 逾期任務 | 紅色邊框 + ⏰ 標記 |
| Edge - 多人同時審批 | row_version 衝突提示 |

## 6. 權限控管

- 承辦人：看到 assigned to 自己的待辦
- 主管：看到 assigned to 自己的簽核待辦 + 所屬單位的待辦
- 任務依 assignee_employee_id 或 assignee_role_id 過濾

## 7. 相關頁面與路由

- 待辦列表：/admin/review-tasks
- 待辦詳情：/admin/review-tasks/[id]（可精簡，直接跳案件詳情）
- 完整案件：/admin/benefits/[id]
