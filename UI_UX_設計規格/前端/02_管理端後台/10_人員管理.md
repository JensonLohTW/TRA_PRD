# 人員管理

## 1. 功能概述

管理端維護員工主檔、眷屬資料，檢視資格歷史與扣繳歷史，支援批次匯入與匯出。

## 2. 頁面架構

### 人員列表（/admin/employees）

```
+------------------------------------------+
|  人員管理                          [+新增]  [匯入] |
+------------------------------------------+
|  篩選列                                  |
|  單位：[全部 v]   狀態：[全部 v]            |
|  關鍵字：[________]                       |
+------------------------------------------+
|  ┌──────┬──────┬──────┬──────┬──────┬──┐  |
|  │員編  │姓名  │單位  │狀態  │資格  │操作│  |
|  ├──────┼──────┼──────┼──────┼──────┼──┤  |
|  │EMP001│王小明│臺北段│在職  │符合  │[編輯]│
|  │EMP002│李四  │板橋段│在職  │不符合│[編輯]│

|  └──────┴──────┴──────┴──────┴──────┴──┘  |
+------------------------------------------+
```

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 人員列表 | DataTable | employee |
| 篩選列 | SearchFilterBar | org_node / employment_status |
| 員工詳情 Tabs | Tabs | employee / dependent / history |
| 基本資料 Form | Form | employee 欄位 |
| 眷屬表格 | Table | employee_dependent |
| 資格歷史表格 | Table | employee_subsidy_eligibility_history |
| 扣繳歷史表格 | Table | employee_payroll_deduction_history |
| 匯入 Button | Button | → /admin/employees/import |
| 匯出 Button | Button | GET /employees/export |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<DataTable>` (自訂) | 人員列表 | - |
| `<SearchFilterBar>` (自訂) | 篩選 | - |
| `<Tabs>` | 員工詳情分頁 | - |
| `<Form>` | 基本資料編輯 | - |
| `<Input>` | 聯絡資訊 | - |
| `<Select>` | 在職狀態/人員類型 | - |
| `<Table>` | 眷屬/歷史列表 | - |
| `<Button>` | 儲存/匯入/匯出 | - |
| `<Pagination>` | 分頁 | - |

## 5. 權限控管

- 系統管理者：完整 CRUD
- 承辦人：檢視所轄範圍員工資料
- 身分證字號僅顯示遮罩值

## 6. 相關頁面與路由

- 人員列表：/admin/employees
- 員工詳情：/admin/employees/[id]
- 批次匯入：/admin/employees/import
