# 組織與權限管理

## 1. 功能概述

管理端維護組織架構樹、角色定義、功能權限映射、資料範圍設定與人員配置。為 RBAC 授權模型的後台管理介面。

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 組織樹 | TreeView | org_node |
| 節點詳情 Form | Form | org_node.node_name, node_code, level_type_id |
| 角色列表 | DataTable | position_role |
| 角色權限設定 | TransferList | role_permission (多對多) |
| 資料範圍設定 | Form + Select | role_data_scope |
| 人員配置列表 | DataTable | employee_position_assignment |
| 配置編輯 Dialog | Dialog | assignment 表單 + branch_responsibility |
| 員工搜尋 Select | Combobox | employee |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<TreeView>` (自訂) | 組織樹 | 展開/收起/拖放排序 |
| `<DataTable>` (自訂) | 角色/人員列表 | - |
| `<Form>` | 節點/角色/配置表單 | - |
| `<Input>` | 名稱/代碼 | - |
| `<Select>` | 層級/角色選擇 | - |
| `<TransferList>` (自訂) | 權限分配 | 左側可用/右側已選 |
| `<Checkbox>` | 功能權限選取 | - |
| `<Dialog>` | 新增/編輯子表單 | - |
| `<Combobox>` (自訂) | 員工搜尋 | 可搜尋 |
| `<Badge>` | 角色類型/狀態 | - |

## 5. 權限控管

- 僅系統管理者可存取組織與權限管理
- 權限異動即時生效並留存軌跡

## 6. 相關頁面與路由

- 組織管理：/admin/org
- 角色管理：/admin/org/roles
- 權限管理：/admin/org/permissions
- 人員配置：/admin/org/assignments
