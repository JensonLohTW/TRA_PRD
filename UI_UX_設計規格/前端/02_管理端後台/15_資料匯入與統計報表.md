# 資料匯入與統計報表

## 1. 功能概述

管理端執行組織/人員/資格/扣繳資料的批次匯入（含錯誤回饋），以及產製各類統計報表（案件統計、撥款統計等）。

## 2. 頁面架構

### 資料匯入（/admin/employees/import）

```
+------------------------------------------+
|  資料匯入                                 |
+------------------------------------------+
|  匯入類型：[人員資料 v]                    |
|  選擇檔案：[選擇檔案] (CSV/Excel)          |
|  上傳格式說明：[下載範本]                  |
|                                          |
|  [開始匯入]                               |
+------------------------------------------+
|  匯入結果                                |
|  ┌──────────────────────────────────────┐ |
|  │ 總筆數：150 | 成功：145 | 失敗：5    │ |
|  │ ┌── 錯誤清單 ──────────────────────┐ │ |
|  │ │ 第 3 行：員編重複                 │ │ |
|  │ │ 第 27 行：Email 格式錯誤          │ │ |
|  │ │ ...                              │ │ |
|  │ └──────────────────────────────────┘ │ |
|  │ [匯出錯誤清單]  [重新匯入]            │ |
|  └──────────────────────────────────────┘ |
+------------------------------------------+
```

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 匯入類型 Select | Select | import_batch.batch_type |
| 檔案上傳 | FileUploader | file_resource → import_batch |
| 匯入結果 | Card | import_batch (success_count/fail_count) |
| 錯誤清單 | Table | import_row_error |
| 匯出錯誤 Button | Button | GET /import/{id}/export-errors |
| 報表類型 Select | Select | 前端定義 |
| 日期區間 Pickers | DatePicker × 2 | range |
| 圖表預覽 | BarChart / PieChart (Recharts) | API 統計資料 |
| 匯出 Button | Button | GET /reports/export?type=X |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<Select>` | 匯入類型/報表類型 | - |
| `<FileUploader>` (自訂) | 上傳檔案 | CSV/Excel 限定 |
| `<Card>` | 匯入結果摘要 | 成功/失敗計數 |
| `<Table>` | 錯誤清單 | - |
| `<Button>` | 匯出/下載範本 | - |
| `<DatePicker>` | 日期區間 | range mode |
| `<BarChart>` (Recharts) | 條形圖 | - |
| `<PieChart>` (Recharts) | 圓餅圖 | - |
| `<Tabs>` | 報表類型切換 | 申請/撥款/人員 |

## 5. 權限控管

- 匯入：限系統管理者
- 報表：承辦人/主管/管理者（依資料範圍過濾）

## 6. 相關頁面與路由

- 資料匯入：/admin/employees/import
- 統計報表：/admin/reports
