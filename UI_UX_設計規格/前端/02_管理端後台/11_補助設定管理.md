# 補助設定管理

## 1. 功能概述

管理端維護補助類型定義、表單版本管理、年度上限規則設定。補助類型採參數化設定，支援動態表單 schema 管理。

## 3. 頁面元素與 DB 欄位對應

| UI 元素 | 組件類型 | API/DB 對應 |
|---------|----------|-------------|
| 類型列表 | DataTable | application_type |
| 類型編輯 Dialog | Dialog | application_type form |
| 表單版本列表 | DataTable | benefit_form_version |
| Schema 檢視 | CodeBlock (JSON) | benefit_form_version.schema_json |
| 版本編輯 Dialog | Dialog | form_version 表單 + JSON 編輯 |
| 上限規則列表 | DataTable | benefit_annual_limit_rule |
| 規則編輯 Dialog | Dialog | limit_mode / limit_value |

## 4. Shadcn UI 組件建議

| 組件 | 用途 | 備註 |
|------|------|------|
| `<DataTable>` (自訂) | 各類列表 | - |
| `<Dialog>` | 新增/編輯表單 | - |
| `<Form>` | 設定表單 | - |
| `<Input>` | 金額/次數 | type="number" |
| `<Select>` | 領域/模式選擇 | - |
| `<Checkbox>` | 需附件/需啟用 | - |
| `<CodeBlock>` (自訂) | Schema JSON 檢視 | syntax highlight |

## 5. 權限控管

- 僅系統管理者可操作補助設定
- 設定異動即時影響前台申請規則

## 6. 相關頁面與路由

- 補助類型：/admin/benefit-settings/types
- 表單版本：/admin/benefit-settings/form-versions
- 年度上限：/admin/benefit-settings/annual-limits
