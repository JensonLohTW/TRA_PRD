# 資料欄位命名對照總表

## 使用說明

本表統整前端 UI 顯示名稱、API JSON 欄位名稱與資料庫欄位名稱的對應關係，確保前後端開發時欄位命名一致。

---

## 1. 共用欄位

| UI 顯示名稱 | API 欄位名 | DB 欄位名 | 型態 | 說明 |
|------------|------------|-----------|------|------|
| 主鍵 ID | id | {table}\_id | bigint | 各表主鍵 |
| 建立時間 | createdAt | created_at | timestamptz | 建立時間 |
| 更新時間 | updatedAt | updated_at | timestamptz | 更新時間 |
| 版本號 | rowVersion | row_version | int | 樂觀鎖 |
| 軟刪除 | isDeleted | is_deleted | boolean | 軟刪除標記 |
| 建立人 | createdBy | created_by | bigint | FK to employee |
| 更新人 | updatedBy | updated_by | bigint | FK to employee |
