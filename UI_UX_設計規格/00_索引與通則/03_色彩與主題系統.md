# 色彩與主題系統

## 1. 品牌色系

### 1.1 主色 — 臺鐵藍

```css
--primary:       #004C97   hsl(210, 100%, 35%)
--primary-light: #3374B0   hsl(210,  55%, 45%)
--primary-dark:  #003366   hsl(210, 100%, 20%)
```

主色用於：主要按鈕、導航選中狀態、連結、重點強調。

### 1.2 輔助色 — 臺鐵金

```css
--accent:        #FFB800   hsl(45, 100%, 50%)
--accent-light:  #FFD666   hsl(45, 100%, 65%)
```

輔助色用於：焦點 Highlight、里程碑標記、VIP 標籤。

### 1.3 背景與表層

```css
--background:    #FAFBFC   hsl(210, 20%, 98%)    /* 頁面背景 */
--card:          #FFFFFF   hsl(0,   0%, 100%)    /* 卡片/表層 */
--muted:         #F0F2F5   hsl(210, 20%, 94%)    /* 禁用/淡化區 */
```

## 4. 暗色模式調色

暗色模式降低對比度，減少長時間使用的眼睛疲勞：

```css
.dark {
  --background:    #1A1D23
  --card:          #23272E
  --muted:         #2D3138
  --border:        #3A3F47
  --primary:       #4A90D9   /* 調亮以便在暗背景可讀 */
  --accent:        #FFC833
}
```
