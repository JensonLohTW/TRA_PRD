# 全域組件庫

## 使用說明

本文件列出跨三大端口共用的自訂組件。每個組件標註其基礎 Shadcn UI 原始組件與擴充功能。

---

## 1. 資料表格 — DataTable

| 項目 | 說明 |
|------|------|
| **基礎組件** | `<Table>` (Shadcn) |
| **擴充** | 排序、篩選、分頁、欄位可見性切換、行選取（Checkbox）、匯出 CSV |
| **Props** | columns, data, pageSize, onRowClick, selectable, loading, emptyMessage |
| **使用場景** | 案件列表、人員列表、批次明細、日誌查詢 |
| **狀態** | loading → `<Skeleton>` row × 5；empty → 插圖 + emptyMessage |

```tsx
// 範例用法
<DataTable
  columns={applicationColumns}
  data={applications}
  pageSize={20}
  onRowClick={(row) => router.push(`/admin/benefits/${row.id}`)}
  selectable
  loading={isLoading}
/>
```

## 2. 動態表單 — DynamicForm

| 項目 | 說明 |
|------|------|
| **基礎組件** | `<Form>` + `<FormField>` (Shadcn) + React Hook Form + Zod |
| **擴充** | 依 schema_json 動態渲染欄位、條件顯示（showWhen）、計算欄位（computed）、附件清單 |
| **Props** | schema: FormSchemaJSON, form: UseFormReturn, onSubmit, readOnly |
| **使用場景** | 八大補助申請表單、補助設定編輯 |
| **支援欄位類型** | text, number, date, select, radio, checkbox, textarea, file, section (群組分隔線) |

---

## 3. 檔案上傳器 — FileUploader

| 項目 | 說明 |
|------|------|
| **基礎組件** | `<Input type="file">` + `<Progress>` (Shadcn) |
| **擴充** | 拖放上傳、多檔上傳、進度條、預覽縮圖、刪除、OCR 狀態指示 |
| **Props** | accept, maxSize, maxFiles, onUpload, onDelete, files, uploadProgress, ocrStatus |
| **使用場景** | 補助申請附件上傳、商店合約上傳、公告附件 |
| **OCR 狀態** | pending → `<Badge>辨識中</Badge>` → completed → `<Badge variant="success">已辨識</Badge>` → error → `<Badge variant="destructive">辨識異常</Badge>` |

---

## 4. 狀態標籤 — StatusBadge

| 項目 | 說明 |
|------|------|

## 5. 流程時間線 — WorkflowTimeline

| 項目 | 說明 |
|------|------|
| **基礎組件** | 自訂（無 Shadcn 對應，基於 Tailwind） |
| **擴充** | 垂直時間線、節點狀態（完成/進行中/待處理/退回）、操作人與時間、審批意見 |
| **Props** | steps: WorkflowStep[], currentStepCode |
| **使用場景** | 申請詳情頁流程進度、案件時間線 |

```tsx
interface WorkflowStep {
  stepCode: string;
  stepName: string;
  status: 'completed' | 'active' | 'pending' | 'rejected';
  actorName?: string;
  actedAt?: string;
  comment?: string;
}
```

## 8. 確認對話框 — ConfirmDialog

| 項目 | 說明 |
|------|------|
| **基礎組件** | `<AlertDialog>` (Shadcn) |
| **擴充** | 支援三種模式：alert（警示）/ confirm（需輸入原因）/ destructive（危險操作） |

| **使用場景** | 核准/退回/駁回確認、批次刪除確認 |
